# Spotik — анализ библиотеки Spotify

Скрипт вытаскивает все плейлисты, подписчиков, треки-дубли (в нескольких плейлистах)
и жанры каждого плейлиста.

## Запуск

```bash
# 1. впиши доступы
cp .env.example .env
# открой .env и вставь Client ID / Secret из developer.spotify.com/dashboard

# 2. запусти (при первом запуске откроется браузер для логина)
./.venv/bin/python analyze.py
```

Результат — в папке `out/`:
- `playlists.csv` — плейлист, владелец, подписчики, кол-во треков, топ-жанры
- `duplicates.csv` — треки, лежащие в >1 плейлисте
- `playlist_genres.csv` — топ-жанры по каждому плейлисту
- `library.json` — всё одним файлом

## Оговорки
- «Подписчики» = `followers.total` плейлиста (сколько людей его сохранили). Сколько раз
  лично ты его открывал/слушал — API не отдаёт.
- У плейлистов в Spotify **нет** поля жанра. Жанр выводится агрегацией жанров артистов
  всех треков. Число в скобках = сколько треков попадает в жанр.
